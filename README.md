# FairAudit AI 🛡️

FairAudit AI is a full-stack application designed to audit AI models for bias using standard fairness metrics. 

## Project Structure
- **/frontend**: React + Vite + Tailwind CSS dashboard.
- **/backend**: FastAPI + Python (Pandas/NumPy) audit engine.

## Features
- User Authentication (JWT)
- Dataset Upload & Audit (CSV/JSON)
- Fairness Visualization (Radar & Bar Charts)
- Automated Fairness Grading (A/B/C/F)
